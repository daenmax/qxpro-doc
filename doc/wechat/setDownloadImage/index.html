    目录

    设置下载图片时间（setDownloadImage）

    # 接口说明

    • 接口路径：http://127.0.0.1:8888/wechat/httpapi
    • 请求方法：POST
    • 功能描述：设置下载图片时间

    # 微信版本支持情况

    提示

    ✅ 3.9.9.34

    ✅ 3.9.10.16

    ✅ 3.9.11.18

    ✅ 3.9.12.16

    ✅ 3.9.12.44

    ✅ 3.9.12.54

    # Body请求参数

    参数 类型 必填 说明
    type string 是 接口type
    data object 是 请求数据对象
    data.type string 是 类型，1=全天下载，2=全天不下载

    # Body请求示例

    {
    "type": "setDownloadImage",
    "data": {
        "type": "1"
    }
}

    # 响应参数

    参数 类型 说明
    code integer 响应码
    msg string 响应消息
    wxid string 微信ID
    port integer 端口
    pid integer 进程ID
    flag string 标志
    timestamp string 时间戳
    result object 结果对象

    # 响应示例

    {
    "code": 200,
    "msg": "操作成功",
    "result": {},
    "wxid": "",
    "port": 8888,
    "pid": 2840,
    "flag": "7888",
    "timestamp": "1716609820642"
}
