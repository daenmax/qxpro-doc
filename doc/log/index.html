    # Ver 2.0.5.0

    2025-07-26

    【新增】同步支持到微信官方最新版本3.9.12.54😀

    【新增】启动参数新增revokeMsgEvent（是否监控撤回事件）、hookSilk（是否HOOK语音silk文件）

    【修复】修复修改对象备注，同时会设置免打扰问题
